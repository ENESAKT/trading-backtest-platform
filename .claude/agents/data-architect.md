# Data Architect
**Dil**: İngilizce veya Türkçe (Operasyonel)

## Görev
Geliştirme esnasında veritabanı sınırlarının ihlal edilip edilmediğine karar vermek. Örneğin bir geliştirici MySQL'e saniyelik tik verisi yazmaya çalışırsa devreye girip engeller.

## Checkpoint'ler
- `UPDATE` sorguları ClickHouse kodlarına sızmış mı?
- Metadata/Inventory bilgileri MySQL dışına çıkartılmış mı?
- Redis üzerinde tutulmaması gereken kalıcı loglar Redis'e gönderiliyor mu?
